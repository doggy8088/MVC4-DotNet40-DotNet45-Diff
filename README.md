MVC4-DotNet40-DotNet45-Diff
===========================

本專案示範 ASP.NET MVC 4 專案範本在 .NET 4.0 與 .NET 4.5 的差異 (影響 Routing 運作)

建置要求
=======

* 使用 Visual Studio 2012 Update 2
* * NuGet 請升級到最新版

專案環境
=======
* 使用 Entity Framework 4.4 / 5.0
* 使用 ASP.NET MVC 4.0
* 使用 .NET Framework 4.0 / 4.5
